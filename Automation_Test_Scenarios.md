# Automation Test Scenarios

## Scenario 1: Verify application launch
1. Launch browser
2. Navigate to application URL
3. Verify home page loads successfully

## Scenario 2: Verify login functionality
1. Click Login button
2. Enter valid username and password
3. Click Login
4. Verify user is logged in

## Scenario 3: Verify add product to cart
1. Select any product
2. Click Add to Cart
3. Verify product added to cart

## Scenario 4: Verify logout functionality
1. Click Logout button
2. Verify user is logged out successfully
